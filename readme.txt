== Description ==
This widget plugin will display a list of posts or pages.

* Displays posts/pages.
* Change sort order.
* Display/hide created or updated date.
* Include / exclude IDs

== Installation ==

1. Upload this directory to the 'wp-content/plugins/' directory.
1. Activate the plugin through the 'Plugins' menu in WordPress.
1. Open 'Widgets' menu, add the 'Post/Page list' widget into the sidebar and choose your options.
